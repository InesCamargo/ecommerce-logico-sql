🛒 Projeto Lógico de Banco de Dados — E-commerce
Este repositório contém o projeto lógico do banco de dados para um cenário de e‑commerce, desenvolvido como parte do desafio da DIO.

🎯 Objetivo
Implementar o modelo lógico considerando os refinamentos solicitados no desafio, incluindo:

Cliente PF ou PJ (não pode ser os dois) 

Pagamento com múltiplas formas de pagamento por pedido 

Entrega com status e código de rastreio 

📌 Arquivos do projeto
script_ddl.sql — criação das tabelas e constraints

script_inserts.sql — dados de teste

queries.sql — consultas SQL incluindo SELECT, WHERE, HAVING, ORDER BY e JOIN

modelo-logico.png — diagrama lógico 

Esse script foi feito em conjunto com a Copilot e conferido por mim. Está totalmente de acordo com a proposta do projeto e fizemos os testes para garantir sua funcionalidade.
